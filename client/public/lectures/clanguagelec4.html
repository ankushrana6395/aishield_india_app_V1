Functions - Code Reusability

Functions C programming ka ek important concept hai. Function ek block of code hota hai jo specific task perform karta hai.

Functions ka use karke aap:

  • Code ko reusable bana sakte hain
  • Complex programs ko chunks mein divide kar sakte hain
  • Code readability improve kar sakte hain
  • Debugging aur testing aasan ho jati hai

Real-life Example: Jaise aap microwave mein khana garam karne ke liye same buttons use karte hain har bar, waise hi functions ko repeatedly call karke same code execute kar sakte hain.

main()
→
calculate()
calculate()

result

Function Calling Process

Function Declaration, Definition, and Call

Function banane ke liye teen steps hote hain:

  1. Function Declaration - Function ka signature batata hai
  2. Function Definition - Actual code implementation
  3. Function Call - Function ko execute karna
// Function declaration (prototype)
int add(int a, int b);

// Function definition
int add(int a, int b) {
    return a + b;
}

// Function call
int result = add(5, 3);

Note: Function declaration optional hai agar aap function definition main function se pehle likhte hain.

Function Components:

  • Return Type: Function jo value return karta hai
  • Function Name: Function ka unique identifier
  • Parameters: Input values function ke liye
  • Function Body: Actual code jo execute hota hai

Arrays - Ek Se Jyada Values Store Karna

Array ek data structure hai jo same type ki multiple values ko ek variable mein store karta hai.

Arrays ka use karke aap:

  • Multiple values ek saath store kar sakte hain
  • Data ko efficiently manage kar sakte hain
  • Loops ke through easily process kar sakte hain

Array declare karne ka syntax:

data_type array_name[size];

Real-life Example: Jaise aap ek egg tray mein multiple eggs store karte hain, waise hi array ek variable mein multiple values store karta hai.

Array Memory Representation

Array Declaration and Initialization

Array ko different ways mein declare aur initialize kiya ja sakta hai:

// Different ways to declare and initialize arrays
// Method 1: Declaration then initialization
int numbers[5];
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;

// Method 2: Declaration with initialization
int numbers[5] = {10, 20, 30, 40, 50};

// Method 3: Size omitted during initialization
int numbers[] = {10, 20, 30, 40, 50};

// Method 4: Partial initialization
int numbers[5] = {10, 20}; // Rest elements will be 0

Yaad rakhein: Array indices 0 se start hote hain. Agar array ka size 5 hai, to indices 0 se 4 tak honge.

Array Access:

int numbers[5] = {10, 20, 30, 40, 50};

// Accessing array elements
printf("%d", numbers[0]); // Output: 10
printf("%d", numbers[2]); // Output: 30
printf("%d", numbers[4]); // Output: 50

// Modifying array elements
numbers[1] = 25; // Now numbers[1] is 25

Arrays and Loops - Powerful Combination

Loops ke through arrays ko process karna efficient hota hai. Aap for loop ka use karke easily array elements ko access aur modify kar sakte hain.

// Example: Array traversal using for loop
int numbers[5] = {10, 20, 30, 40, 50};
int sum = 0;

// Calculate sum of array elements
for (int i = 0; i < 5; i++) {
    sum += numbers[i];
}

printf("Sum: %d\n", sum); // Output: Sum: 150

// Print array elements in reverse
for (int i = 4; i >= 0; i--) {
    printf("%d ", numbers[i]);
}
// Output: 50 40 30 20 10

Tip: Array size ko variable mein store karna acha practice hai, taki aap easily array size change kar saken.

Loop Through Array:

// Find maximum element in array
int arr[] = {45, 12, 67, 23, 89};
int max = arr[0];
int size = 5;

for (int i = 1; i < size; i++) {
    if (arr[i] > max) {
        max = arr[i];
    }
}

printf("Maximum element: %d\n", max); // Output: 89

Functions and Arrays Together

Functions mein arrays ko arguments ke roop mein pass kiya ja sakta hai. Jab aap array ko function mein pass karte hain, to actually array ka base address pass hota hai.

// Function to calculate array sum
int calculateSum(int arr[], int size) {
    int sum = 0;
    for (int i = 0; i < size; i++) {
        sum += arr[i];
    }
    return sum;
}

int main() {
    int numbers[] = {10, 20, 30, 40, 50};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    int total = calculateSum(numbers, size);
    printf("Sum: %d\n", total); // Output: 150
    return 0;
}

Note: Jab aap array ko function mein pass karte hain, to function array ki copy nahi banta, balki original array par kaam karta hai. Isliye agar function mein array modify karenge to original array change hoga.

Array Passing Mechanism:

// Function to modify array elements
void doubleValues(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        arr[i] *= 2; // Modify original array
    }
}

int main() {
    int nums[] = {1, 2, 3, 4, 5};
    int size = 5;

    doubleValues(nums, size);

    // Original array is modified
    for (int i = 0; i < size; i++) {
        printf("%d ", nums[i]); // Output: 2 4 6 8 10
    }
    return 0;
}
