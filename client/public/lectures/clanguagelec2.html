Keywords (Reserved Words)

Keywords wo special words hote hain jo C language mein already defined hote hain. Inka pre-defined meaning hota hai aur inhe variable names ya function names ke liye use nahi kar sakte.

Examples: int, float, char, if, else, for, while, do, switch, case, break, continue, return, etc.

Real-life Example: Jaise traffic signals ke colors ka pre-defined meaning hota hai - red means stop, green means go. Waise hi keywords ka pre-defined meaning hota hai.

Note: C language mein total 32 keywords hote hain. Inhe aap variable name ke liye use nahi kar sakte.

// Valid variable names
int age = 20;
float salary = 50000.50;

// Invalid - keyword can't be variable name
// int int = 10; // ERROR
// float return = 5.5; // ERROR

Variables - Chuney

Variable ek named storage location hoti hai jahan aap data store kar sakte hain. Variable banane ke liye aapko uska data type aur name specify karna hota hai.

Variable declare karne ka syntax:

data_type variable_name;

Variable initialize karne ka syntax:

data_type variable_name = value;

Rules for variable names:

  • Variable name letters, digits, aur underscore (_) se start ho sakta hai
  • Variable name case sensitive hota hai (age aur AGE alag hain)
  • Variable name keyword nahi ho sakta
  • Variable name space contain nahi kar sakta
// Variable declaration and initialization
int age; // Declaration
age = 20; // Assignment

float height = 5.9; // Initialization
char grade = 'A';

// Multiple variables
int a = 5, b = 10, c = 15;

Input lena - scanf() Function

scanf() function ka use user se input lene ke liye hota hai. Ye stdio.h header file mein defined hota hai.

Syntax:

scanf("format_specifier", &variable);

Format specifiers:

  • %d - integer ke liye
  • %f - float ke liye
  • %lf - double ke liye
  • %c - character ke liye
  • %s - string ke liye

Yaad rakhein: Variable se pehle & (address of operator) lagana zaroori hai. Ye batata hai ki value kahan store karni hai.

#include <stdio.h>

int main() {
    int age;
    float height;
    char grade;

    printf("Enter your age: ");
    scanf("%d", &age);

    printf("Enter your height: ");
    scanf("%f", &height);

    printf("Enter your grade: ");
    scanf(" %c", &grade);

    printf("Age: %d, Height: %.2f, Grade: %c\n", age, height, grade);
    return 0;
}

Complete Example - Poorna Udaharan

Chalo ek complete program dekhte hain jisme hum variables declare karte hain, user se input lete hain, aur output display karte hain:

#include <stdio.h>

int main() {
    // Variable declarations
    int num1, num2;
    float result;
    char operator;

    // Input from user
    printf("Enter first number: ");
    scanf("%d", &num1);

    printf("Enter operator (+, -, *, /): ");
    scanf(" %c", &operator);

    printf("Enter second number: ");
    scanf("%d", &num2);

    // Calculation
    if (operator == '+') {
        result = num1 + num2;
    } else if (operator == '-') {
        result = num1 - num2;
    } else if (operator == '*') {
        result = num1 * num2;
    } else if (operator == '/') {
        result = (float)num1 / num2;
    } else {
        printf("Invalid operator!\n");
        return 1;
    }

    // Output result
    printf("Result: %.2f\n", result);
    return 0;
}

Program Output:

Enter first number: 15
Enter operator (+, -, *, /): *
Enter second number: 4
Result: 60.00

Special Note: Character input lene se pehle agar koi aur input liya hai to " %c" mein space use karein. Yeh extra newline character ko consume kar leta hai.
