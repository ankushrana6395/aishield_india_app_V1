File Handling - Introduction

File Handling C programming ka ek important concept hai jisme hum files ko read, write, aur manipulate karte hain. Files ka use data ko permanently store karne ke liye hota hai.

File handling ke liye C mein standard library functions stdio.h mein available hote hain.

File handling ke main steps:

  1. File open karna
  2. File read/write operations perform karna
  3. File close karna

Real-life Example: Jaise aap notebook mein data likhte hain aur baad mein padhte hain, waise hi file handling mein hum computer ki files mein data store karte hain aur baad mein access karte hain.

File Opening Modes

File open karne ke liye fopen() function use hota hai, jisme file name aur mode specify karte hain.

Syntax:

FILE *fopen(const char *filename, const char *mode);

Common file opening modes:

"r"

Read mode. File must exist.

"w"

Write mode. Creates file if doesn't exist, truncates if exists.

"a"

Append mode. Creates file if doesn't exist, appends if exists.

"r+"

Read/Write mode. File must exist.

"w+"

Read/Write mode. Creates file if doesn't exist, truncates if exists.

"a+"

Read/Append mode. Creates file if doesn't exist.

Yaad rakhein: File open karne ke baad hamesha check karein ki file successfully open hui ya nahi. Agar file open nahi hoti to fopen() NULL return karta hai.

// File opening example
#include <stdio.h>

int main() {
    FILE *file;
    
    // Open file for writing
    file = fopen("example.txt", "w");
    
    // Check if file opened successfully
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    // File operations here
    
    // Close the file
    fclose(file);
    return 0;
}

Writing to Files

Files mein data write karne ke liye kai functions available hain:

Function Description Usage
fprintf() Formatted output to file fprintf(file, "Text %d", var);
fputs() String output to file fputs("Text", file);
fputc() Character output to file fputc('A', file);
fwrite() Binary data output fwrite(data, size, count, file);

Note: fprintf() printf() function jaisa hi hota hai, lekin ye file mein output likhta hai screen par nahi.

// Writing to file example
#include <stdio.h>

int main() {
    FILE *file;
    
    file = fopen("data.txt", "w");
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    // Write different types of data
    fprintf(file, "Hello, World!\n");
    fputs("This is a line of text.\n", file);
    fputc('A', file);
    fprintf(file, "\nNumber: %d\n", 42);
    
    fclose(file);
    printf("Data written to file successfully!\n");
    return 0;
}

Reading from Files

Files se data read karne ke liye kai functions available hain:

Function Description Usage
fscanf() Formatted input from file fscanf(file, "%d", &var);
fgets() String input from file fgets(str, size, file);
fgetc() Character input from file char c = fgetc(file);
fread() Binary data input fread(data, size, count, file);

Yaad rakhein: File read karte waqt EOF (End of File) check karna important hai. feof() function use karke check kar sakte hain ki file ka end aa gaya hai ya nahi.

// Reading from file example
#include <stdio.h>

int main() {
    FILE *file;
    char buffer[100];
    
    file = fopen("data.txt", "r");
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    // Read file line by line
    printf("File contents:\n");
    while (fgets(buffer, sizeof(buffer), file) != NULL) {
        printf("%s", buffer);
    }
    
    fclose(file);
    return 0;
}

File Positioning Functions

File positioning functions allow aap file mein different positions par ja kar read/write operations perform kar sakte hain.

Function Description Usage
fseek() File pointer ko specific position par set karta hai fseek(file, offset, whence);
ftell() Current file position return karta hai long pos = ftell(file);
rewind() File pointer ko file ke beginning par set karta hai rewind(file);
feof() Check karta hai ki file ka end reached hai ya nahi if (feof(file))

Note: fseek() mein whence parameter SEEK_SET (beginning), SEEK_CUR (current position), ya SEEK_END (end of file) ho sakta hai.

// File positioning example
#include <stdio.h>

int main() {
    FILE *file;
    long position;
    
    file = fopen("example.txt", "r+");
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    // Get current position
    position = ftell(file);
    printf("Current position: %ld\n", position);
    
    // Move to end of file
    fseek(file, 0, SEEK_END);
    position = ftell(file);
    printf("End position: %ld\n", position);
    
    // Go back to beginning
    rewind(file);
    position = ftell(file);
    printf("After rewind: %ld\n", position);
    
    fclose(file);
    return 0;
}
